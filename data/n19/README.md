# n19

Dataset for the `n=19` base configuration.

## Files

Inputs:

- `lines.csv` — floating-point line data;
- `omatrix.json` — the specified O-matrix.

Certificates (generated by the verifier):

- `interval_certificate.json` — geometric realization certificate;
- `combinatorial_certificate.json` — combinatorial properties certificate.
